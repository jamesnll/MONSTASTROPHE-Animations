# MONSTASTROPHE-Animations

A collection of weapon rigs and animations created in Blender for use in my upcoming game project **MONSTASTROPHE**. This repo serves as both an asset library and a showcase of the rigging and animation work done across a variety of firearms.

---

## 📁 File Structure

```
MONSTASTROPHE-Animations/
├── Assault Rifles/
│   ├── Animations/
│   ├── Meshes/
│   ├── Rigs/
│   ├── Textures/
│   └── _Exported/
├── Others/
│   ├── Animations/
│   ├── Meshes/
│   ├── Rigs/
│   ├── Textures/
│   └── _Exported/
├── Pistols/
│   ├── Animations/
│   ├── Meshes/
│   ├── Rigs/
│   ├── Textures/
│   └── _Exported/
├── Rifles/
│   ├── Animations/
│   ├── Meshes/
│   ├── Rigs/
│   ├── Textures/
│   └── _Exported/
├── Shotguns/
│   ├── Animations/
│   ├── Meshes/
│   ├── Rigs/
│   ├── Textures/
│   └── _Exported/
└── Submachine Guns/
    ├── Animations/
    ├── Meshes/
    ├── Rigs/
    ├── Textures/
    └── _Exported/
```

Each weapon category follows the same internal structure:

| Folder | Contents |
|---|---|
| `Animations/` | Keyframed action data (idle, fire, reload, etc.) |
| `Meshes/` | Original weapon mesh geometry & author credits |
| `Rigs/` | Armature and bone setups |
| `Textures/` | Material and texture maps |
| `_Exported/` | Game-ready exported files |
